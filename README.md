# Cinematix

A modern movie discovery application built with Next.js that allows users to explore trending films, search for movies, and view detailed information using a movie database API.

---

## Preview

![Project Preview](./public/preview.png)

---

## Overview

This project provides a fast and responsive interface for browsing movies. It focuses on performance, clean UI design, and a smooth user experience using modern web technologies.

---

## Features

- Movie search functionality
- Trending and popular movies listing
- Detailed movie information pages (overview, ratings, release date, etc.)
- Responsive design for all screen sizes
- Optimized performance with Next.js App Router

---

- Next.js

## Installation

# Fill in your keys in .env.local
